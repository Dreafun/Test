# 🌙 Scarlet Notes

Это пример **сайта статей и гайдов** в более “живом” стиле — не как сухая документация, а ближе к аккуратному блогу.

## Что это даёт

- красивые статьи с нормальной типографикой
- тёмную тему
- боковую навигацию
- поиск по сайту
- блоки советов, предупреждений, вкладки, таблицы и код
- удобную структуру под гайды, лорные заметки и подборки

```{tip}
:class: dropdown

**Куда это лучше подходит**

Если тебе нужен сайт именно для статей, а не только для техдоков,
такой вариант обычно ощущается приятнее.
```

## Что внутри проекта

- `guides/moon_library.md` — пример атмосферной статьи
- `guides/formatting_playground.md` — сборка с форматированием
- `mkdocs.yml` — главный конфиг сайта
- `requirements.txt` — зависимости

## Быстрый запуск локально

```bash
pip install -r requirements.txt
mkdocs serve
```

После этого сайт откроется локально, обычно по адресу `http://127.0.0.1:8000`.

## Сценарии использования

### Для гайдов
- инструкции
- пошаговые статьи
- ответы на частые вопросы

### Для текстов
- атмосферные заметки
- лорные материалы
- архив статей

### Для проекта
- разделы по темам
- подборки материалов
- “база знаний”, которая выглядит не скучно
