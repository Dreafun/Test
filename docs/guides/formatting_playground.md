# 🧪 Полигон форматирования

Ниже — краткая, но насыщенная сборка полезного форматирования для сайта статей.

## Заголовки

# H1
## H2
### H3
#### H4

## Выделение текста

- **Жирный**
- *Курсив*
- ==Маркер==
- ~~Зачёркнутый~~
- `Код внутри строки`

## Цитата

> Иногда одна короткая цитата задаёт настроение всей статье.

## Списки

### Обычные
- пункт один
- пункт два
  - подпункт

### Нумерованные
1. шаг один
2. шаг два
3. шаг три

### Задачи
- [x] написать черновик
- [ ] добавить картинки
- [ ] вычитать текст

## Спойлер / сворачивание

??? note "Нажми, чтобы открыть"
    Внутри может быть дополнительная информация, которую не хочется держать на виду постоянно.

## Вкладки

=== "Windows"

    1. Открой папку проекта.
    2. Запусти терминал.
    3. Введи команду сборки.

=== "Linux"

    1. Открой терминал.
    2. Перейди в каталог проекта.
    3. Выполни `mkdocs serve`.

=== "Телефон"

    1. Открой GitHub.
    2. Найди нужный `.md` файл.
    3. Отредактируй текст прямо в браузере.
